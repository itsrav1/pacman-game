# 🎮 Pac-Man Inspired Game (Unity Project)

> A classic Pac-Man-inspired game developed using Unity, focusing on recreating the original arcade experience with smooth gameplay, intelligent enemy behavior, and an engaging scoring system.

---

## ✨ About the Project

This project is a recreation of the iconic **Pac-Man** game built using the Unity engine.  
It focuses on implementing core game development concepts such as movement mechanics, AI behavior, collision detection, and game logic design.

The goal was to deliver a smooth and fun playable experience while strengthening fundamental Unity development skills.

---

## 🕹️ Gameplay Features

- 👾 Classic Pac-Man-style movement system  
- 👻 Enemy AI behavior and path interaction  
- 🍒 Score system with collectibles  
- 💥 Collision detection between player, enemies, and objects  
- 🎯 Smooth and responsive controls  

---
